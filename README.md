# VibroVarioAuto v1.5b

**Автоматический вариометр для параплана на ESP32 Watchy**

Вибрация + звук (Brauneiger-style) + комплементарный фильтр (IMU+Baro) + энергосбережение.

Форк/порт оригинального [VibroVario](https://github.com/isemaster/VibroVario) от isemaster.
Оригинал — ESP32 Watchy, BMP390, BMA423, E-Ink 1.54".

![Watchy](media/watchy.jpeg)

## Возможности

- ✅ **Беззвучный режим** — вибромотор на запястье
- ✅ **Звук Brauneiger** — частота и ритм пропорциональны Vz (подключаемый буззер)
- ✅ **Комплементарный фильтр** — акселерометр + барометр. Мгновенный отклик, никакого дрейфа. Работает при любой ориентации часов
- ✅ **E-Ink дисплей** — читаем на солнце, 0 мВт в покое
- ✅ **Умное энергосбережение** — без движения спит сутками, с движением — просыпается раз в минуту
- ✅ **Автостарт** — сам замечает взлёт (Vz > 1 м/с) и начинает запись
- ✅ **Автостоп** — через 5 мин стабильной высоты останавливает запись
- ✅ **Вес** 24 г (с батареей), **полёт** 10+ часов

## Кнопки

```
         SETUP            UP
          ●────────────────●
          │                │
          │    ЭКРАН       │
          │    200×200     │
          │                │
          ●────────────────●
          OK             DOWN
```

| Кнопка | GPIO | Название | На экране часов | В полёте | В настройках |
|--------|------|---------|----------------|----------|--------------|
| Верх-лев | 26 | SETUP | — (питание сенсоров) | — | — |
| Верх-прав | 25 | UP | — | Выйти в часы | Выйти в часы |
| Ниж-лев | 4 | OK | Старт полёта | Стоп полёта | Toggle ON/OFF |
| Ниж-прав | 35 | DOWN | Настройки | — | Сброс в ON |

## Сон и пробуждение

| Условие | Поведение |
|---------|-----------|
| Нет движения | Глубокий сон. Пробуждение только по кнопке UP |
| Есть движение, нет набора высоты | Сон 60 сек, проверка 1 замер BMP. Счётчик бездействия 15 мин → сон 24ч |
| Vz > 0.75 м/с за минуту | Автостарт полёта (показывает FLIGHT! на экране) |
| Vz < 0.75 и нет движения 5 мин | Автостоп записи, возврат в сон |
| DOWN на экране часов | Вход в настройки (Buzzer ON/OFF, Vibro ON/OFF) |

## Состояния автомата (State Machine)

Устройство управляется конечным автоматом (FSM) с 5 состояниями:

```
    ┌──────────┐
    │  CLOCK   │ ◄──── глубокий сон/пробуждение
    └────┬─────┘
         │
    ┌────▼──────┐   ┌────────────┐
    │ SETTINGS  │   │ CALIBRATING│
    └────▲──────┘   └─────┬──────┘
         │                │
    ┌────┴──────────┐     │
    │   RUNNING     │◄────┘
    │ (vario active)│
    └──────┬───────-┘
           │
    ┌──────▼───────-┐
    │   STOPPED     │
    │ (stats frozen)│
    └──────┬───────-┘
           │
      (back to CLOCK)
```

### Transition Table

| Текущее | Действие | Следующее | Описание |
|---------|----------|-----------|----------|
| CLOCK | OK (press[1]) | CALIBRATING → RUNNING | Ручной старт полёта |
| CLOCK | DOWN (press[0]) | SETTINGS | Открыть настройки |
| CLOCK | UP (press[2]) | (deep sleep) | Уход в сон (24ч) |
| CLOCK | Vz > порога (авто) | FLIGHT! приглашение → OK/DOWN | Обнаружение взлёта |
| SETTINGS | OK | SETTINGS (toggle) | Buzzer ON/OFF → Vibro ON/OFF |
| SETTINGS | DOWN | SETTINGS (reset) | Включить всё обратно |
| SETTINGS | UP | CLOCK | Выход в часы |
| CALIBRATING | (2 сек) | RUNNING | Калибровка завершена |
| CALIBRATING | (ошибка) | CLOCK | Калибровка не удалась |
| RUNNING | OK | STOPPED | Пауза, показать статистику |
| RUNNING | UP | CLOCK | Экстренный выход (варио стоп) |
| RUNNING | Автопосадка (5 мин) | CLOCK | Vz < 0.75 + нет движения |
| STOPPED | OK | CLOCK | Сброс RTC на 00:00 |
| STOPPED | DOWN | CALIBRATING → RUNNING | Новый полёт |
| STOPPED | UP | CLOCK | Выход в часы |
| (deep sleep) | UP btn wake | CLOCK | Пробуждение по кнопке |
| (deep sleep) | RTC alarm | CLOCK | Проверка барометра |
| (deep sleep) | BMA motion | CLOCK | Пробуждение по движению |

## Подключение буззера

```
Buzzer+ → GPIO 32 (через транзистор если >50 мА)
Buzzer− → GND
```

Watchy V2: GPIO 32 свободен.
Watchy V1/V1.5: замени на 16 или 17.
